Tecnologias que vamos usar em nossos encontros:


    Acode


    É uma IDE (Integrated Development Environment) - ambiente de desenvolvimento integrado para dispositivos Android, está disponível gratuitamente na Google Playstore.





    TERMUX


    É um ambiente Linux para aparelhos Android, permitindo trabalharmos com linhas de comando, NPM, Git, dentre outras funções interessantes. Com ele, vamos ser capazes de desenvolver aplicativos usando tecnologias como React, NodeJS, e até mesmo integrar com o Acode!



    Atenção! Para a instalação correta do Termux, sigam este passo-a-passo. Não é recomendado instalar esse aplicativo pela Playstore, por estar desatualizado.



    Chegando nessa página, vamos ter duas opções: o arm64 e armeabi. Vamos instalar de acordo com a arquitetura de nosso dispositivo. Temos a opção universal também, todavia, é mais pesado (101 MB)





    Em seguida, vamos pesquisar a arquitetura de nosso aparelho. No meu caso, é o arm64, então vou instalar o termux arm64.
