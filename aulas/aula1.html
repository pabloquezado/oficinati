    HTML

    É uma linguagem de marcação responsável por definir a estrutura web.


    Sua sintaxe (estrutura) se dá por tags com blocos de abertura <> e fechamento </>


    Estrutura básica do arquivo .html com exemplos de tags:


    
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title> Título </title>
</head>

<body>

<h1> Cabeçalho </h1>
<p> Parágrafo </p>
            
</body>
</html>


    Resultado

    Não se preocupem com todas essas tags da estrutura básica logo de início, o importante é rodar o código no seu editor. Pensem na estrutura básica como algo que sempre acompanha seu código.

    Observem que a maioria das tags inicia com <> e fecha com </> 

    <h1> Cabeçalho </h1>
<p> Parágrafo </p>

    Imaginem as tags do HTML como sanduíches. As chaves de abertura e fechamento seriam o pão, enquanto o conteúdo seria o queijo.

    <pão>
queijo
</pão>

    Tá vendo só como a estrutura do HTML é fácil?

    Certo. Já entendi como as tags funcionam, mas e se eu quiser deixar tudo mais bonito?


    É aí que a gente entra com o CSS!



    CSS


    Conhecido como Cascade Style Sheet (folha de estilo em cascata), possui a função de estilizar nossa estrutura já definida pelas tags do HTML.


    Sua extensão de arquivo é .css e podemos chamá-la de 3 formas diferentes: em linha, interna ou externa.


    Para facilitar o aprendizado, vamos usar o inline - para usarmos CSS direto no HTML sem necessidade de linkar para um arquivo externo.

    <style>
Inserir CSS aqui
</style>

    Mãos na massa!


    Vamos começar pela estrutura básica do HTML. Dentro do Acode, na seção de plu-gins, vocês vão instalar o "snippets"



    Essa extensão vai completar seu código e fechar as tags automaticamente, além de permitir atalhos.


    Com o snippets instalado, criem um novo arquivo -> nome.html (no lugar de nome, pode ser qualquer um!)




    Agora, digitem "html" e cliquem na segunda opção

    Ele vai te gerar a estrutura para trabalharmos com HTML.


    Estilização


    A maneira mais simples de estilizar um elemento HTML é definindo uma classe a ele e, no CSS, aplicar a estilização que desejamos.


    Para ficar mais claro, vamos adicionar 4 parágrafos e dar a eles uma classe chamada "cor"

    <p class="cor"></p>
<p class="cor"></p>
<p class="cor"></p>
<p class="cor"></p>

    Dentro do <body> temos, então, nosso código completo:

    
  <!DOCTYPE html>
  <html lang="en">
  <head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title> Título </title>
  </head>
  
  <body>
  
  <p class="cor"> Parágrafo </p>
  <p class="cor"> Parágrafo </p>
  <p class="cor"> Parágrafo </p>
  <p class="cor"> Parágrafo </p>
              
  </body>
  </html>

    Criamos 4 parágrafos e associamos a classe 'cor' a eles.

    Agora, entramos com o CSS:

      .cor {
        color: red;
      }

    No CSS, chamamos a classe que definimos no HTML através do sinal de ponto "."

    Em seguida, colocamos { } e a estilização que desejarmos.

    Ao final de cada estilização, devemos fechar com ponto e vírgula ;


    Certo, mas como vou adicionar esse CSS no HTML, se a estrutura do HTML é diferente?


    Vamos usar a tag <style> para conectar o CSS ao HTML

    
  <!DOCTYPE html>
  <html lang="en">
  <head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title> Título </title>
  </head>
  
  <body>
  
  <p class="cor"> Parágrafo </p>
  <p class="cor"> Parágrafo </p>
  <p class="cor"> Parágrafo </p>
  <p class="cor"> Parágrafo </p>

  <style>
  .cor{
    color: red;
  }
  </style>
              
  </body>
  </html>

    O tamanho do texto está muito pequeno, não acham? Vamos adicionar um pouco mais de tamanho.

      .cor {
        color: red;
        font-size: 30px;
      }


    Definimos o tamanho do texto para 30 pixels. Com CSS, podemos manipular tamanho, posição, cor, dentre outros atributos.



    Classes e ID


    Agora, vamos supor que eu queira deixar esses 4 textos de cor vermelha, mas apenas um deles seja diferente.

    Mas como fazer isso, sem precisar criar outra classe?


    Vamos usar o ID, que permite atribuição única ao elemento.

    <p class="cor" id="unico"> Parágrafo </p>
<p class="cor"> Parágrafo </p>
<p class="cor"> Parágrafo </p>
<p class="cor"> Parágrafo </p>

    No CSS, usamos # para chamar o ID.

      .cor{
  color: red;
  font-size: 30px;  
  }

  #unico{
    color: blue;
  }


    Com o ID, fomos capazes de alterar a cor de apenas um parágrafo, mesmo ele pertencendo a classe que lhe confere a cor vermelha, o ID modificou sua cor, destacando-o dos demais.


    Esse recurso é muito útil quando usamos classes e desejamos alterar apenas um ou dois elementos.
